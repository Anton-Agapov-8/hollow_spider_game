	1.3) Усовершенсвовать алгоритм, добавив прорисовку пола на карте.
	1.4) Добавить текстуры для стен.
	1.5) Добавить коллизию стен и игрока (стены твёрдые).

	1.7) Добавить прорисовку лазера при выстреле.

		1.8.3) Улучшить алгоритм так, чтобы при осмотре с разных сторон враги-спрайты имели разные текстуры (вид сбоку, спереди, сзади)
		1.8.4) Добавить ИИ для врагов: они будут перемещаться по карте, а огда увидят игрока, то будут либо атаковать, либо убегать. Атаковать - если здоровья у врага много или рядом много союзников. Иначе нужно убегать.
	1.9) Добавить кнопки на карту. Каждая кнопка будет кодироваться с помощью цвета обозначающего её на карте-изображении
2) Сделать главный экран для игры.

5) Нарисовать карты уровней, которые будут запускаться с помощью движка. Добавить запуск этих карт в игру.
